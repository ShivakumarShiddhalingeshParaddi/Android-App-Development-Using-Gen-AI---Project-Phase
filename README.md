Namma Railu Buddy

Namma Railu Buddy is an Android application developed using Generative AI to assist users with railway-related queries and services. The app aims to provide a smart, user-friendly platform for accessing train information efficiently.

🔹 Features
🤖 AI-based railway query system
📅 Train schedules and information
📍 User-friendly interface
⚡ Fast and responsive performance
📱 Works across Android devices
🔹 Technologies Used
Java / Kotlin
Android Studio
Generative AI APIs
Firebase (optional backend)
🔹 Objective

To create a digital platform that enhances user experience in accessing railway information and supports self-employment opportunities through technology.

🔹 Future Enhancements
Live train tracking
Ticket booking integration
Voice-based AI assistant
